# GitHub Security

The repository includes:

- Product CI
- Dependency Review
- CodeQL for Python and GitHub Actions
- OpenSSF Scorecard
- Dependabot
- CODEOWNERS

If GitHub code scanning is disabled for the repository, CodeQL SARIF upload may fail until code scanning is enabled in repository settings.
