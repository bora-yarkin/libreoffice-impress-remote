Control LibreOffice Impress from your phone by scanning a QR code.

LibreOffice Impress Remote turns a phone into a presenter remote for LibreOffice Impress. Start the remote from the Slide Show menu, scan the displayed QR code, and control the presentation in the phone's web browser. No mobile app is required.

The phone interface provides current-slide preview, presenter notes, previous and next controls, tap-to-advance, total-presentation and current-slide timers, first slide, last slide, go-to-slide controls, and fullscreen slide mode where supported by the browser.

The default and recommended connection mode is the local network, suitable for devices on the same Wi-Fi network or personal hotspot. Optional LocalTunnel, direct IPv6, and self-hosted relay modes are included as experimental alternatives for more difficult network environments.
